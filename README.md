# ITMO_PRACTICE_2_COURSE
**«Разработка системы сбора и визуализации полётных данных для мультиагентных систем»**

Также некоторая информация лежит в файле program_structure.txt

## Используемое оборудование

* БПЛА Pioneer базовый
* Модуль Radxa Zero 3W
* Компьютер под управлением Windows/Linux
* Python 3.12-14
* Библиотека Pioneer SDK2

---

## Подготовка к работе

Перед запуском программы необходимо выполнить следующие действия:

1. Подключить аккумулятор к БПЛА.
2. Дождаться появления Wi-Fi сети вида `RaZero-...`.
3. Подключиться к сети Wi-Fi.

Параметры подключения:

* Имя сети: `RaZero-...`
* Пароль: `geoscan123`


Для взаимодействия с автопилотом через Pioneer SDK2 используется TCP-соединение:

```text
10.42.0.1:20556
```

---

## Подключение по SSH

Для корректной передачи команд автопилоту необходимо установить SSH-соединение с модулем Radxa.
Можно обойтись и без SSH, но скорее всего он понадобится для того, чтобы поменять айпи и порты
дронам в пределах одного роутера

Команда подключения:

```bash
ssh pioneer@10.42.0.1
```

Параметры авторизации:

```text
Логин: pioneer
Пароль: geoscan123
```

### Примечание для Windows

Если при подключении появляется сообщение:

```text
Connection closed by 10.42.0.1 port 22
```

необходимо:

1. Установить OpenSSH Server.
2. Включить правило брандмауэра:

```text
OpenSSH-Server-In-TCP
```
