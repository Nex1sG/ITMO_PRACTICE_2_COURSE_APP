
## Описание программы

Программа выполняет следующие действия:

1. Подключается к БПЛА через Pioneer SDK2.
2. Подписывается на событие запуска двигателей (`ENGINE_STARTED`).
3. Отправляет команду запуска двигателей (`arm()`).
4. Ожидает 3 секунды.
5. Останавливает двигатели командой `disarm()`.
6. Закрывает соединение с автопилотом.

---

# Экземпляр класса Pioneer

```python
pioneer = Pioneer()
```

## Описание класса:
Класс Pioneer является основным интерфейсом взаимодействия с библиотекой

### Аргументы класса

| Аргумент | Тип данных | Значение по умолчанию | Описание аргумента |
|----------|------------|----------------------|-------------------|
| serial | str | _ | Подключаемый последовательный порт |
| tcp | str | 127.0.0.1:20556 | TCP адрес и порт |
| baudrate | int | 57600 | Скорость передачи данных |
| wait_callback | bool | True | Флаг включения блокирования методов на ожидание выполнения callback-функций |
| safety_command | bool | True | Флаг включения проверки разрешенных команд в зависимости от состояния полета |
| logger | bool | True | Флаг включения логирования |

---

## arm(timeout=5, retries=0)

**Описание метода:**  
Запуск моторов квадрокоптера

**Аргументы метода:**
- `timeout` - задержка перед запуском моторов
- `retries` - количество повторных попыток запуска моторов

**Тип данных:**  
bool

**Возвращает:**  
True, если команда успешно выполнена, иначе False

**Примечание:**  
Ожидает события ENGINES_STARTED при параметре Pioneer.wait_callback=True

---

## disarm()

**Описание метода:**  
Выключение моторов квадрокоптера

**Тип данных:**  
bool

**Возвращает:**  
True, если команда успешно выполнена, иначе False

## Принцип работы программы

После создания объекта `Pioneer` библиотека устанавливает TCP-соединение с автопилотом.

Функция:

```python
pioneer.arm()
```

отправляет команду запуска двигателей.

При успешном запуске автопилот генерирует событие:

```text
ENGINE_STARTED
```

на которое подписана пользовательская функция:

```python
test_of_subscribe_function()
```

После завершения проверки двигатели останавливаются командой:

```python
pioneer.disarm()
```

а соединение закрывается:

```python
pioneer.close_connection()
```
